# Saucedemo Automation Este projeto é um conjunto de testes automatizados para o site Saucedemo. 
# Utiliza a linguagem de programação Java e a biblioteca de testes JUnit para garantir a funcionalidade e a estabilidade do site. 
## Estrutura do Projeto - `src/test/java`: Contém os arquivos de teste em Java. - `src/main/java`: 
## Contém os arquivos de suporte e utilitários. - `src/resources`: Contém arquivos de configuração e dados de teste. 

## Pré-requisitos - Java JDK 8 ou superior - Maven 

## Instalação 
## 1.Clone o repositório para sua máquina local: ```bash git clone https://github.com/ederbueno/saucedemo142 ``` 
## 2. Navegue até o diretório do projeto: ```bash cd saucedemo142 ``` 
## 3. Instale as dependências do projeto usando o Maven: ```bash mvn install ``` 

## Execução dos Testes Para executar os testes, utilize o seguinte comando: ```bash mvn test
